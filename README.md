# 📌 Overview

Saraha is a secure anonymous messaging backend application that allows users to receive anonymous messages from others while maintaining privacy and security.

The project follows a scalable modular architecture with authentication systems, validation middleware, secure token handling, and clean service-controller separation.

# 🚀 Installation

## 1️⃣ Clone Repository

```bash id="u7x6yl"
git clone https://github.com/YousefHDev/saraha-api.git
```

---

## 2️⃣ Install Dependencies

```bash id="lcyl43"
npm install
```

---

## 3️⃣ Configure Environment Variables

Create `.env` file:

```env id="6t8b0q"
PORT=
MONGO_URI=
TOKEN_SIGNATURE=
ENCRYPTION_SIGNATURE=
```

---

## 4️⃣ Run Development Server

```bash id="u4ps1f"
npm run dev
```
